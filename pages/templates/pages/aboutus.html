About This Project

This project is a demonstration of an online tech shop, showcasing features for a seamless shopping experience. It is designed to provide users with the ability to browse, purchase, and manage their accounts in a user-friendly environment.

Payment Integration

This demo project is connected to Stripe for handling payments. Users can test the payment functionality using the following test card details:

Successful Payment

Use this card to simulate a successful payment transaction.

  • Card Number: 4242 4242 4242 4242
  • Expiry Date: Any valid future date (e.g., 12/34)
  • CVC: Any three digits (e.g., 123)

Note: This is for testing purposes only and will not process real transactions.

Declined Payment

Use this card to simulate a declined payment transaction.

  • Card Number: 4000 0000 0000 0002
  • Expiry Date: Any valid future date (e.g., 12/34)
  • CVC: Any three digits (e.g., 123)

Note: This card simulates a declined payment response.

Not Enough Funds

Use this card to simulate a transaction with insufficient funds.

  • Card Number: 4000 0000 0000 9995
  • Expiry Date: Any valid future date (e.g., 12/34)
  • CVC: Any three digits (e.g., 123)

Note: This card simulates a payment failure due to insufficient funds.

Shopping Cart

Customers can browse products and add them to their shopping cart. However, to proceed with the payment process, customers must log in to their account. This step ensures the security of the transaction and enables order confirmation via email.

Order Confirmation

Once a purchase is completed, an email will be sent to the customer with the order details. This ensures transparency and provides a record of the transaction for the customer.

Account Creation

To use this demo, customers are required to create an account with a valid email address. After registration, users must confirm their email address to access all features. This step is crucial for testing the authentication and account management functionality.
